Format:
http://ip:port/{programName}/{version}/{aipName}/{parameters}


1. Get area list
Request:
/getAreaList

Response:
{areas:["name":"gaoxin"] 
}

2. Get price list
Request
/getPriceList

{prices:["10000-12000"]
}
